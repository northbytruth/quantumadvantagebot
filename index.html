Native GitHub Actions Integration

To run resource telemetry and cost estimation directly inside your repository on every pull request, drop this workflow file into your project layout at .github/workflows/quantum-ci.yml:

name: Quantum Infrastructure Guard

on:
  pull_request:
    branches: [ main ]

jobs:
  analyze-quantum-code:
    runs-on: ubuntu-latest
    steps:
      - name: Checkout Client Codebase
        uses: actions/checkout@v4

      - name: Run Quantum Advantage Analysis
        uses: YOUR_GITHUB_USER/quantum-advantage@v1
        with:
          target-file: 'src/circuits/simulation.qs'
          # Leave blank for Free Community Tier, or apply your signed PQC token
          license-signature: ${{ secrets.QUANTUM_ADVANTAGE_LICENSE_KEY }}
